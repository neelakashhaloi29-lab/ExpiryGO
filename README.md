# ExpiryGO

## Overview

ExpiryGO is an offline-first Progressive Web App (PWA) that helps users track product expiry dates, reduce food waste, and manage household inventory.

---

## Current Features

- Dashboard
- Add Product
- Edit Product
- Delete Product
- Categories
- Local Storage
- Offline Support (PWA)
- Responsive Design

---

## Planned Features

- Barcode Scanning
- Receipt OCR
- AI Recipe Suggestions
- Firebase Cloud Sync
- Shared Family Inventory
- Notifications
- Food Waste Analytics
- Smart Shopping List

---

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Local Storage
- PWA

---

## Run Locally

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
