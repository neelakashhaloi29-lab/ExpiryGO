# ExpiryGO - AI Context

## Project

ExpiryGO is an offline-first expiry tracker built using Vite + React.

The project is intended to become a production-quality application.

---

## Current Tech Stack

- React
- Vite
- JavaScript
- CSS
- Local Storage
- Progressive Web App (PWA)

Firebase will be added later.

---

## Current Features

- Dashboard
- Add Product
- Edit Product
- Delete Product
- Product Categories
- Expiry Calculations
- Local Storage
- Offline Support
- Responsive Layout

---

## Current Bug

The HTML5 date input occasionally interprets dates incorrectly.

Example:

Cheese product should be

2026-07-02

but becomes

2026-02-07

Investigate the edit form and date parsing.

---

## Development Rules

- Never recreate the project.
- Never reinstall React or Vite.
- Never overwrite working code.
- Inspect the codebase before changing anything.
- Fix any bug before starting a new feature.
- Keep changes small.
- Explain every change.
- Stop after completing each feature.
- Wait for approval before the next feature.

---

## Current Goal

Fix the date handling bug.

After that continue building the application incrementally.
