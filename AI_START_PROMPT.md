# AI Startup Instructions

Before making ANY changes:

1. Read these files completely:
   - AI_CONTEXT.md
   - PROJECT_RULES.md
   - ROADMAP.md
   - CHANGELOG.md

2. Inspect the existing codebase.

3. Explain:
   - What currently works.
   - What is incomplete.
   - Any bugs found.

4. Do NOT modify any code until you explain your plan.

5. Build only ONE feature at a time.

6. After each completed feature:
   - Verify it works.
   - Fix any bugs.
   - Update CHANGELOG.md.
   - Stop and wait for approval.

Never:
- recreate the project
- reinstall React
- reinstall Vite
- overwrite working code
- delete existing features
- leave the project in a broken state
